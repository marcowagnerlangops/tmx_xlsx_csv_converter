# tmx_xlsx_csv_converter

A lightweight tool for converting TMX files to human-readable `.xlsx` and `.csv` formats — and back again.

> **Note on language codes:** When converting from XLSX or CSV back to TMX, you may need to adjust language codes to match the exact format expected by your enterprise system.

## Usage

1. **Choose conversion direction** — select whether you want to convert TMX → XLSX/CSV or XLSX/CSV → TMX.
2. **Select source file** — browse to and select the file you want to convert.
3. **Choose output destination** — specify where the converted file should be saved.
4. **Click Convert** — run the conversion. That's it!
